# 미니 프로젝트 : 숙박 예약 API 서비스 
## 🖥 프로젝트 개요
> 1. 프로젝트 주제 :  숙박 예약 웹서비스 구현 프로젝트
> 2. 프로젝트 기간 : 11월 20일(월) ~ 12월 01일(금)
> 3. 주요 목표 : Spring Boot, DB 설계, DB 트랜잭션, RESTful API 설계, 테스트 코드 작성, Spring Security 적용
